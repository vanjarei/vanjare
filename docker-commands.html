Top 20 Docker Commands

  1. docker run: Create and start a new container. It's used to launch a container from a Docker image.
  2. docker build: Build an image from a Dockerfile. This command is used to create a Docker image from a Dockerfile.
  3. docker pull: Pull an image or a repository from a registry. It fetches Docker images from a registry like Docker Hub.
  4. docker push: Push an image or a repository to a registry. It uploads Docker images to a registry like Docker Hub.
  5. docker ps: List containers. It displays information about running containers.
  6. docker exec: Run a command in a running container. This command executes a command inside a running container.
  7. docker images: List images. It shows a list of Docker images stored locally on the system.
  8. docker stop: Stop one or more running containers. It halts the execution of a running container.
  9. docker rm: Remove one or more containers. This command deletes one or more stopped containers.
  10. docker rmi: Remove one or more images. It deletes Docker images from the local image cache.
  11. docker-compose up: Create and start containers defined in a docker-compose.yml file. It starts services defined in a Docker Compose file.
  12. docker-compose down: Stop and remove containers created by 'docker-compose up'. It stops and removes containers created by Docker Compose.
  13. docker logs: Fetch the logs of a container. This command retrieves the logs of a specific container.
  14. docker inspect: Return low-level information on Docker objects. It provides detailed information about Docker objects like containers, images, networks, etc.
  15. docker network ls: List networks. It displays a list of Docker networks on the system.
  16. docker volume ls: List volumes. It lists Docker volumes created on the system.
  17. docker restart: Restart one or more containers. It restarts one or more stopped containers.
  18. docker-compose build: Build or rebuild services defined in a docker-compose.yml file. It builds Docker images for services defined in a Docker Compose file.
  19. docker-compose start: Start services defined in a docker-compose.yml file. It starts services defined in a Docker Compose file.
  20. docker-compose stop: Stop services defined in a docker-compose.yml file. It stops services defined in a Docker Compose file.
