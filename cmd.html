Network Commands in Windows

1. ping

Purpose: Tests connectivity between your system and a remote host (IP address or domain).

Command:

ping <hostname or IP address>

Example:

ping google.com
ping 8.8.8.8

2. tracert

Purpose: Traces the path packets take from your system to a remote host.

Command:

tracert <hostname or IP address>

Example:

tracert google.com

3. ipconfig

Purpose: Displays current network configuration (IP address, subnet mask, default gateway, etc.).

Command:

ipconfig

Example:

ipconfig /all
ipconfig /release
ipconfig /renew

4. netstat

Purpose: Displays active connections, listening ports, and network statistics.

Command:

netstat

Example:

netstat -an
netstat -aon
netstat -b

5. nslookup

Purpose: Queries DNS to retrieve IP address or other information for a domain.

Command:

nslookup <hostname>

Example:

nslookup google.com

6. route

Purpose: Displays or modifies the system's routing table.

Command:

route print

Example:

route print

7. telnet

Purpose: Connects to a remote system or service using the Telnet protocol (useful for testing open ports).

Command:

telnet <hostname> <port>

Example:

telnet google.com 80

8. netsh

Purpose: Configures and manages network settings.

Command:

netsh interface ip show config

Example:

netsh interface ip show config

9. arp

Purpose: Displays or modifies the ARP cache (mapping of IP addresses to MAC addresses).

Command:

arp -a

Example:

arp -a

10. ipconfig /flushdns

Purpose: Clears the DNS cache to resolve DNS resolution issues.

Command:

ipconfig /flushdns

Example:

ipconfig /flushdns

11. net view

Purpose: Displays a list of networked computers in a workgroup or domain.

Command:

net view

Example:

net view

12. net use

Purpose: Connects to or disconnects from shared network resources (e.g., mapped network drives).

Command:

net use <drive letter>: \\<hostname>\\<shared folder>

Example:

net use z: \\192.168.1.100\sharedfolder

13. systeminfo

Purpose: Displays detailed information about the system configuration, including network information.

Command:

systeminfo

Example:

systeminfo

14. PowerShell: Test-NetConnection

Purpose: Tests the network connectivity to a remote host, similar to ping but more feature-rich.

Command:

Test-NetConnection -ComputerName <hostname or IP> -Port <port>

Example:

Test-NetConnection -ComputerName google.com -Port 80

15. netstat -b

Purpose: Displays the executable associated with each active connection (requires admin rights).

Command:

netstat -b

Example:

netstat -b

16. net start

Purpose: Lists all running network services.

Command:

net start

Example:

net start

17. wmic nicconfig get

Purpose: Retrieves detailed information about network adapter settings.

Command:

wmic nicconfig get

Example:

wmic nicconfig get Description, IPAddress, DefaultIPGateway

18. netsh wlan show interfaces

Purpose: Displays detailed information about the Wi-Fi connection (signal strength, IP address, etc.).

Command:

netsh wlan show interfaces

Example:

netsh wlan show interfaces

19. netsh advfirewall

Purpose: Configures Windows Firewall settings.

Command:

netsh advfirewall show allprofiles

Example:

netsh advfirewall show allprofiles

20. netsh interface ipv4 set address

Purpose: Configures IPv4 settings, including IP address, subnet mask, and gateway.

Command:

netsh interface ipv4 set address name="<InterfaceName>" static <IP> <SubnetMask> <Gateway>

Example:

netsh interface ipv4 set address name="Ethernet" static 192.168.1.100 255.255.255.0 192.168.1.1

21. netstat -an

Purpose: Displays all active network connections and their respective statuses.

Command:

netstat -an

Example:

netstat -an

22. nmap (third-party tool)

Purpose: A network exploration tool for scanning open ports and discovering hosts (not natively available in Windows).

Command:

nmap <IP address or hostname>

Example:

nmap 192.168.1.1

23. netsh interface set interface

Purpose: Enables or disables a network interface.

Command:

netsh interface set interface "<InterfaceName>" admin=disable

Example:

netsh interface set interface "Ethernet" admin=disable

24. hostname

Purpose: Displays or sets the hostname (computer name).

Command:

hostname

Example:

hostname

25. Get-NetAdapter (PowerShell)

Purpose: Retrieves information about network adapters.

Command:

Get-NetAdapter

Example:

Get-NetAdapter
